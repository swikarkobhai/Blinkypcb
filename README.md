Blinking LED PCB Board

Overview

This project is a simple blinking LED circuit implemented on a custom PCB. It demonstrates the full workflow from schematic design to PCB layout and basic firmware control.

You can use this project as a starting point to learn PCB design, component selection, and embedded programming.

Features

- Custom PCB design
- LED blinking at a fixed interval
- Simple and easy to understand circuit
- Beginner-friendly hardware project

Components

- Microcontroller or timer IC (depending on design)
- LED
- Resistors
- Capacitors
- Power source (battery or USB)

Files Included

- Schematic design files
- PCB layout files
- Gerber files for manufacturing


How It Works

The circuit drives an LED that turns on and off at a fixed interval. This is achieved using either:

- A microcontroller running a simple loop with delay, or
- A timer-based circuit (such as a 555 timer)

Getting Started

1. Clone the repository
2. Review the schematic and PCB design files
3. Order the PCB using the provided Gerber files
4. Assemble the components on the board
5. Upload the firmware if a microcontroller is used
6. Power the board and observe the LED blinking

Applications

- Learning PCB design
- Electronics practice project
- Teaching basic embedded systems
- Prototype for simple signaling systems
